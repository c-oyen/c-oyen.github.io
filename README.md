# c-oyen.github.io

Personal portfolio site for **c-oyen** — built with [Deno](https://https://deno.com) [Astro](https://astro.build) + TailwindCSS.

🐈🟧 Orange cat energy, chaotic curiosity, still shipping things.

## Stack

- **Framework**: [Astro](https://astro.build)
- **Javascript Runtime**: [Deno](https://deno.com)
- **Styling**: TailwindCSS v4 + custom CSS variables
- **Fonts**: Syne (display) + Space Grotesk (body) + JetBrains Mono (code)
- **Deploy**: GitHub Pages via GitHub Actions

## Dev

```bash
deno install
deno task dev       # localhost:4321
deno task build     # builds to dist/ , you add flag --site & --base for the url too
```
