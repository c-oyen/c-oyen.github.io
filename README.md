# c-oyen.github.io

Personal portfolio site for **c-oyen** — built with [Astro](https://astro.build) + TailwindCSS.

🐈🟧 Orange cat energy, chaotic curiosity, still shipping things.

## Stack

- **Framework**: [Astro](https://astro.build)
- **Styling**: TailwindCSS v4 + custom CSS variables
- **Fonts**: Syne (display) + Space Grotesk (body) + JetBrains Mono (code)
- **Deploy**: GitHub Pages via GitHub Actions

## Dev

```bash
npm install
npm run dev       # localhost:4321
npm run build     # builds to dist/
```

## Deploy

Push to `main` — GitHub Actions handles the rest automatically.

> Make sure GitHub Pages is set to use **GitHub Actions** as the source in repo Settings → Pages.
